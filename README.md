# 🏦 Banking Management System

> A complete banking management system developed in **C++** using **Object-Oriented Programming (OOP)** principles. The project simulates the core operations of a real banking system while emphasizing clean architecture, modular design, and file-based data persistence.

---

# 📖 Overview

The **Banking Management System** is a console-based application that provides separate interfaces for **Employees** and **Customers**.

The project was built to strengthen object-oriented programming concepts by implementing a realistic banking environment with multiple banking operations and secure data management.

Instead of using a database, all information is stored permanently using **file-based storage**, allowing the application to preserve data between executions.

# 💾 Data Persistence

Instead of relying on a relational database, the application stores all data inside files.

Stored information includes:

- 👤 Customer Accounts
- 👨‍💼 Employee Accounts
- 💰 Account Balances
- 🔐 Login Credentials
- 🏦 Banking Records

This allows the system to retain all information after closing and reopening the application.
